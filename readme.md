# Jumpline Revival

![el jumplijne](ASSET/normal.svg)

Jumpline Revival is an unofficial revival of the first Jumpline, adding new features and QoL stuff along the way

## Play
You can play the current build of Jumpline Revival on:
- [jumpplus.pages.dev](https://jumpplus.pages.dev)
- [github pages](https://gdplayer1035.github.io/jumpplus)

## Implemented as of now:
- Websim polyfill (to get game running at all)
